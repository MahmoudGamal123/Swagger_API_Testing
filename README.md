# Swagger API Documentation for Pet Store

This repository contains the Swagger API documentation for the Pet Store system. It includes API definitions for managing pets, shops, stores, and orders.

## Available Endpoints

- **/pet**: Manage pet information (add, update, delete pets)
- **/shop**: Handle shop details and inventory
- **/store**: Manage store locations and stock
- **/order**: Process customer orders and track status

## How to Use

1. Clone this repository.
2. Import the Swagger definition into your API client (e.g., Swagger UI or Postman).
3. Explore and test the endpoints as needed.
